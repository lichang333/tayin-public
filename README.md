# 踏印 · 一地一印

中国制县等级（経県値）地图 —— 踏足一地，盖一枚印，积木拼出真实省级轮廓，记录你到访每地的深浅等级。

**在线体验**：https://lichang333.github.io/tayin-public/

## 玩法
- 首页是全国省级图，点省缩放进对应分区（云贵川渝 / 东三省 / 江浙沪皖 …）
- 点州市选到访等级：住过 5 · 住宿 4 · 玩过 3 · 途经 2 · 落地 1 · 未踏 0
- 带 ▦ 角标的可细分到县 / 区（云贵川渝到县、大湾区到区/堂区）
- 登录后印记云端同步；不登录存本机
